# flutter_training

Flutter: un SDK (Software Developpment Kit) developpé par Google permettant de creer des applications
Android, IOS, WEB, Desktop (Macos, Linux, Windows).

Avantages de Flutter:
1. Cross-platform: Un seul code pour plusieurs plateformes
2. Hot Reload: Les modifications apparaissents instantanément
3. Performance elevee: Flutter compile le code en native
4. UI modere: Widget riche et personnalisables
5. Productivite rapide: Developpment rapide pour un MVP (produit minimun viable)
MVP: C'est la premiere version fonctionnelle d'un produit numerique.

Creer un projet Flutter
flutter create --org io.grouv --platforms android,ios project_name
